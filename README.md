## 基于Django+Simpleui+Hexo的博客系统

---

如果觉得对你有帮助的话，请给个star吧

#### 相关技术

前端：hexo-theme-matery，博客主题

后台：SimpleUi （简单配置就会自动生成后台）

Django:Django3.2.0

数据库：Mysql

Python：Python3.8
